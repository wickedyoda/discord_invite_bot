# Country Code Commands

Allows members to append/remove country suffixes on nicknames.

## Commands

- Set: `/country US` or `!country US`
- Clear: `/clear_country` or `!clearcountry`

## Format

- Nickname suffix format: ` - CC`
- Country code is normalized to uppercase two-letter format.

## Notes

- Existing country suffix patterns are stripped before applying a new one.
- Requires permission to manage nicknames.
