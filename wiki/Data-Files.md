# Data Files

Persistent runtime files are stored under `DATA_DIR` (default `data/`).

## Files

- `access_role.txt`
- `role_codes.txt`
- `invite_roles.json`
- `tag_responses.json`
- `firmware_seen.json`
- `web_users.json`
- `bot.log`

## Purpose

- Invite/code role mapping state
- Tag response map
- Firmware seen-entry cache
- Web admin user accounts
- Runtime logging output
